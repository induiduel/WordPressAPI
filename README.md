## Wordpress API
##### Implementation
- Add the JitPack repository to your root build.gradle file

```gradle
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

- Add the dependency

```gradle
dependencies {
	        implementation 'com.github.induiduel:wordpressapi:1.0.1'
	}
```
