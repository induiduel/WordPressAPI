## Wordpress API
##### Implementation
- Add the JitPack repository to your root build.gradle file

```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

- Add the dependency

```
dependencies {
	        implementation 'com.github.induiduel:wordpressapi:1.0.1'
	}
```
